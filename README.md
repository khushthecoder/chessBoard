MAD Hackathon


**Requirements**

  ● Support two-player local mode: White vs Black on the same device.
  
  ● Show a standard 8×8 chessboard with correct starting positions and clear piece
  icons/colors.
  
  ● Enforce legal moves for all pieces (pawn, rook, knight, bishop, queen, king) and
  allow only valid moves for the selected piece.
  
  ● Handle captures, turn switching, and show whose turn it is (“White to move” / “Black
  to move”).
  
  ● Detect check and basic checkmate and display the game result (e.g., “White wins”,
  “Black wins”).
  
  ● Provide New Game and Undo Last Move actions.

**Tasks Completed**

  ● Supports two-player local mode: White vs Black on the same device on opposite directions.

  ● A standard 8×8 chessboard with correct starting positions and clear piece
  icons/colors.
  
  ● When a piece is pressed it can be selected and disselected , it highlightens which piece is selected.
